Upute:
Projekt mora biti izrađen u tehnologijama koje ćemo učiti tijekom semestra:
•	MVC, Blazor, MAUI
•	Web APIs
•	Web Forms, WPF
Frontend aplikacije ne može biti Javascript framework (React, Angular…). Fokus je na .NET tehnologiju i backend.
Izvorni kod aplikacije spremiti na Github.
Tijek izrade aplikacije mora biti vidljiv – “commit” za svaku veću promjenu.
Izraditi dokumentaciju 3-5 stranica koju šaljete na pastabil@vsite.hr
U obzir se uzima:
•	prezentacija projekta,
•	složenost,
•	odgovori na postavljena pitanja
•	kreativnost
Specifikacija:
U dokumentaciji opisati:
•	kako namjestiti aplikaciju na računalo,
•	dijagram arhitekture aplikacije i ER baze (npr. draw.io, Lucidchart),
•	opis što se nalazi u pojedinom modulu projekta,
•	korištene tehnologije,
•	karakteristike aplikacije - screenshotovi i/ili kratki opis korištenja aplikacije
Smjernice:
Arhitektura i razvoj
•	Primjena MVC, MVVM, Clean Architecture ili višeslojne arhitekture
•	Korištenje middleware pipelinea (npr. logging, error handling)
•	Korištenje Dependency Injection (DI)
Sigurnost
•	JWT ili cookie autentikacija
•	Role-based autorizacija (npr. korisnik/admin)
•	Validacija inputa
API dizajn i performanse
•	RESTful Web API dizajn
•	Swagger/OpenAPI dokumentacija
•	Caching (MemoryCache, DistributedCache)
•	Paginacija, sortiranje, filtriranje
Testiranje
•	Jedinični testovi (xUnit, NUnit, Moq)
•	Integracijski testovi (poželjno)
CI/CD
•	Osnovni CI/CD pipeline (GitHub Actions, Azure DevOps)
•	Automatski build/test procesi
Napredne funkcije (ovisno o projektu):
•	Integracija plaćanja (npr. Stripe, PayPal)
•	Upload slika/dokumenata (lokalno ili cloud)
•	Slanje emaila (MailKit, SendGrid)
•	Real-time funkcionalnost (SignalR)
•	Višejezičnost (localization)
